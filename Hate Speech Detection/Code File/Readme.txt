Model Training:
1. Install the libraries in the requirements.txt file

2. Change the input path to the source of the dataset and saved_path to where your saved pickle model has to be stored in the notebook

3. run all cells

Model Deployment:
1. Install the libraries in the requirements.txt file

2. Change the saved_path in prediction_final.py file to the path where the Saved model folder located in your local machine.

3. In command prompt change the directory to Code FIle and Run the code as >> python app.py
